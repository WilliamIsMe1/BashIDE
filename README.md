# BashIDE

This IDE is made to be completely in bash as a superior version of VSCode, fully integrated into all project types, including those from IntelliJ or Gradle or Maven. This is specifically built for Java only. No extensions. 

There will be a file browser tab with dropdown capabilities, a terminal pane for accessing the shell, and an editor pane with the following capabilities:
- Full LSP support
- Intellisense
- Refactoring capabilities
- Syntax highlighting
- Auto-indents
- Auto-brackets
- Code folding
- Right-click-ish functions
- Vim keybinds option
- Emacs option
